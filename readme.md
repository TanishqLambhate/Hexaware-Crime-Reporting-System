# Setup

## Open powershell terminal 

## Virtual ENV
 
```bash
python -m venv myenv
```
 
## Activate
```sh
.\myenv\Scripts\Activate.ps1
```
## Install pyodbc for database connectivity
```sh
pip install pyodbc
```
## Install tabulate for getting output in tabular format
```sh
pip install tabulate
```
## Command to run code
```sh
python main.py
```
